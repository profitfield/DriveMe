# DriveMe Backend

## Описание
Бэкенд для сервиса заказа премиальных автомобилей с водителем.

## Требования
- Node.js 18+
- PostgreSQL 14+
- Redis

## Установка
```bash
# Установка зависимостей
npm install

# Копирование .env файла
cp .env.example .env

# Настройка .env
# Отредактируйте .env файл, указав настройки БД и другие параметры

# Запуск миграций
npm run migration:run

# Установка пароля для админа
npm run set-admin-password
```

## Запуск
```bash
# Разработка
npm run start:dev

# Продакшен
npm run build
npm run start:prod
```

## API Документация
После запуска сервера, Swagger документация доступна по адресу:
http://localhost:3000/api

## Основные функции
- Авторизация через Telegram
- Заказ автомобилей
- Управление водителями
- Административная панель
- Статистика и отчеты
```
